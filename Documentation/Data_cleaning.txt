* Number of rows

* Memory usage

* Missing values

## 4. Categorical Columns

# Date Handling Notes

Depending on the source system, dates may arrive in different formats.

Examples

* `%d/%m/%Y`
* `%m/%d/%Y`

Useful Pandas options

```python
pd.to_datetime(
    column,
    format="mixed",
    dayfirst=True
)
```

or

```python
pd.to_datetime(
    column,
    format="%d/%m/%Y"
)
```

or

```python
pd.to_datetime(
    column,
    format="%m/%d/%Y"
)
```

After conversion, standardize the output using

```python
dt.strftime("%d/%m/%Y")
```

or

```python
dt.strftime("%m/%d/%Y")
```

depending on business requirements.
